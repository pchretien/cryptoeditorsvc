Local Storage:
All your data and your master password are encrypted on your computer and stored on your disk. The data is NEVER stored clear.

Server Backup:
Since the data is ALWAYS encrypted on your computer, there are no clear text data stored on the server. Your password is not stored on the server as well. If you lose your password, there is no way to get it back. The synchronization between the server and CryptoEditor is always performed on your computer.

Multiple Users:
Many user accounts can be created using the same installation of CryptoEditor. You will need one registration per user for the synchronization service.
